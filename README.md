# PCB Design Portfolio - React Version

A modern React portfolio website for iTEDDA, a PCB Design Expert.

## Features

- **Modern React Architecture**: Built with functional components and hooks
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in effects and interactive hover animations
- **Contact Form**: Functional contact form with state management
- **Professional Design**: Clean, modern UI with circuit-themed background

## Sections

1. **Hero Section**: Eye-catching introduction with animated gradient text
2. **Services**: Six key services offered with hover effects
3. **Portfolio**: Recent projects showcase with interactive cards
4. **About**: Professional background, skills, and statistics
5. **Contact**: Functional contact form with validation

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project directory
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

1. Start the development server:
   ```bash
   npm start
   ```

2. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

To create a production build:

```bash
npm run build
```

This will create an optimized build in the `build` folder.

## Customization

### Colors
The main color scheme uses:
- Primary: `#00ff88` (Neon Green)
- Secondary: `#00d4ff` (Cyan)
- Background: Dark gradient from `#0a0a0a` to `#1a1a2e`

### Content
To customize the content:
1. Edit the data arrays in each component
2. Update the personal information in the components
3. Modify the contact information in the Footer component
